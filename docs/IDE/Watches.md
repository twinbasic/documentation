# Watches

When a project isn't open this will be empty.

![Watches](Images/Watches.png "Watches")

Once you open a project it will other buttons available.

![Watches](Images/Watches_1.png "Watches")

If you wish to add a new watch click ![Add](Images/Add.png "Add") and type in your "Expression to watch"

![Watches](Images/Watches_2.png "Watches")

You can remove any with the Clear button ![Clear](Images/Clear.png "Clear")
